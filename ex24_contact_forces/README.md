# ex24_contact_forces: 접촉력 감지 (Contact Forces)

이 예제는 로봇 팔이 물체와 접촉할 때 발생하는 법선 힘(Normal Force)을 측정하고 이를 제어에 활용하는 기초를 학습합니다.

## 📝 설명
- **터치 센서(`touch`):** 특정 사이트(`site`)에 전용 터치 센서를 부착하여, 해당 면에 가해지는 물리적인 힘(N)을 실시간으로 측정함.
- **물체 상호작용:** 푸셔(빨간색)가 자유롭게 놓인 박스(파란색)를 밀 때, 충돌 엔진이 계산한 접촉력을 센서 데이터로 읽어옴.
- **상태 변화 관찰:** 터미널을 통해 푸셔의 타겟 위치에 따라 박스와 닿는 순간 힘이 급격히 증가하고, 박스가 밀려나면서 힘이 변화하는 과정을 관찰할 수 있음.

## 📊 결과 분석
프로그램을 실행하면 왕복 운동하는 푸셔와 박스의 접촉에 따라 다음과 같은 결과가 나타납니다:
- **접촉 전 (`Force: 0.00 N`):** 푸셔와 박스가 떨어져 있을 때는 힘이 0으로 유지됨.
- **충격 순간 (`Force: > 100 N`):** 푸셔가 박스에 부딪히는 찰나에 높은 피크 힘이 발생함.
- **지속 접촉 (`Force: 10~50 N`):** 푸셔가 박스를 계속 밀고 있을 때는 박스의 질량과 마찰력에 비례하는 일정한 항력이 센서에 기록됨.
